Date Picker

Attach [blocDatePickerTrigger] to any element to open a calendar panel. Works with ngModel, formControl, and formControlName.

Basic usage

Default

A plain button that opens the calendar on click.

Custom element

Any element can be a trigger — here a styled <div>.

{{ pickerDiv.displayValue() || 'Choose date' }}

Date Range Picker

Attach [blocDateRangePickerTrigger] to any element to select a date range. Click twice to pick from and to — order doesn't matter, auto-sorted. Works with ngModel, formControl, and FormGroup.

Basic range

Default

Click the first date, then click the second — range is auto-sorted.

Range — Template-driven (ngModel)

Two-way binding

from: {{ ngModelRange.from ? ngModelRange.from.toLocaleDateString() : 'null' }}   to: {{ ngModelRange.to ? ngModelRange.to.toLocaleDateString() : 'null' }}

Range — Reactive forms

FormControl

from: {{ rangeCtrl.value?.from ? rangeCtrl.value!.from!.toLocaleDateString() : 'null' }}   to: {{ rangeCtrl.value?.to ? rangeCtrl.value!.to!.toLocaleDateString() : 'null' }}

FormGroup

Pass a FormGroup with from / to controls — they update individually.

from control: {{ rangeFormGroup.get('from')?.value ? rangeFormGroup.get('from')!.value!.toLocaleDateString() : 'null' }}   to control: {{ rangeFormGroup.get('to')?.value ? rangeFormGroup.get('to')!.value!.toLocaleDateString() : 'null' }}

Range — Min / Max date

Constrained range
