Tooltip

Hover/focus tooltip with intelligent repositioning and delay control, built on the overlay platform layer.

Basic Usage

Simple Tooltip

Attach [blocTooltip] to any element. Hover or focus to trigger.

Disabled Tooltip

Set [tooltipDisabled]="true" to suppress the tooltip entirely.

Positions

All Placements

Use tooltipPosition to control placement: top, bottom, left, right.

Delay Control

Custom Show / Hide Delay

tooltipShowDelay and tooltipHideDelay accept milliseconds.

Custom Tokens

CSS Token Override

Override --bloc-tooltip-* tokens to customize appearance per-instance or globally.
