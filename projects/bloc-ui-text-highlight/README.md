# @bloc-ui/text-highlight

> **Latest:** v1.0.2

Lightweight Angular text-highlighting utility component used to emphasize query matches inside rendered content.

**[Live Documentation & Demos](https://ui.bloc-verse.com/text-highlight)**

---

## Installation

```bash
npm install @bloc-ui/text-highlight
```

**Peer dependencies:** `@angular/common` and `@angular/core` >= 17.

---

## Usage

```ts
import { BlocTextHighlightComponent } from '@bloc-ui/text-highlight';
```
