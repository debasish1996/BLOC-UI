# @bloc-ui/theme

> **Latest:** v0.0.4

Optional theme layer for [**@bloc-ui/core**](https://www.npmjs.com/package/@bloc-ui/core) — provides CSS custom properties, colour tokens, and dark-mode support.

**[Live Documentation & Demos](https://debasish1996.github.io/BLOC-UI/)**

## Install

```bash
npm install @bloc-ui/theme
```

## Usage

Import the theme stylesheet in your application's global styles:

```scss
@use '@bloc-ui/theme/styles/bloc-theme';
```
